          # 动画的应用场景

          属性变化则用 transition 动画, 样式属性发生变化时才触发动画, 比如透明度的变化, 用户拖动滚动条 , 我们则根据拖动位置去改变页面控件的显示属性, 根据属性来设置动画

          animation 动画, 无需触发的动画 , 帧动画, 一般用在无需用户触发或者循环显示的动画上

          # transition 动画, 属性动画

          html 控件要赋予动画的时机是这个 html 控件发生变化

          比如, 我们给一个 div 定义了宽度, 在某种时候又给这个 div 设置了不同的宽度, 为了让这个变化不显得突兀就需要用到动画

          这就是 transition 来定义的动画 , 比如, 当用户点击了某个按钮, 我们就把页面上的某个区域的内容突出显示出来,

          把这个内容区域的宽度变大 , 初始时候这个内容区域可能是 A 样式, 当点击按钮后, 我们通过 js 给它设置了 B 样式 ,

          那么这个时候, 就可以在它的样式里来设置 transition 动画 , 绝大多数我们需要用的是 transition 动画

          目前涉及到的最复杂的可能就是 scroll 相关的 , 当 scroll 到某个屏幕区域, 我们对控件的 style 或 class 进行设置, 外观产生变化 ,

          同时使用 transition 动画属性, 从而让控件的相关属性变化有动态过渡的变化效果

          # animation 命名帧动画

          animation 动画, 帧动画, 命名对话, 指的都是帧动画,为什么叫帧动画呢,因为它是@keyframes 名称 来给动画命名的,

          无需用户主动触发动画, 可以直接通过 css 设置某个 html 控件上应用该动画,

          页面加载后该动画直接执行,而不是样式属性发生变化时才触发动画

          tansition 动画能满足绝大多数的需求, 但有的时候, 如果你希望一个页面控件, 无需主动触发, 它没有什么属性的改变就自发性的在变化

          就需要用到 animation 定义的帧动画,

          比如我们有个圆形 logo 我们希望页面打开后就会自动在那慢慢旋转, 就可以使用帧动画来实现, 比如 react 的初始化页面的 logo 就是这种帧动画

          # transition 动画的使用

          transition 动画是针对 css 样式的变化,来产生过渡效果动画, 所以我们需要对 html 控件设置不同的样式,

          但不可能对一个 html 控件同时设置两个宽度, 一般场景是, 默认情况下设置一个样式, 在某种情况下(比如用户操作点击, 滚动等等),

          我们对这个 html 控件设置不同的样式或者 class , 于是, 我们可以使用 transition 来说明针对这个不同的属性样式, 我们希望产生什么样的过渡效果

          那么常用的 transition 属性有

          transition-property: background ; 最重要的属性, 指定针对哪个 css 属性产生过渡效果

transition-duration: 500ms ; 过渡动画的持续时间

transition-timing-function: 指定过渡动画的时间和变化率

transition-delay: 2s; 指定动画延时多久后才开始

          常见的 css 变化属性, 一般这些 css 属性的变化才能使用动画, 不可能针对所有的 css 属性都能使用动画, 比如 position: absolute 和 position: relative 这种变化是无法进行动画过度的吧!!

          // 常见的用来设置动画的 css 属性

          注意: 位置的变化, 一般不是去改变他的 top , left 属性, 而是用 transform 来设置,

          比如向上移动 100px , 那么用的 css 属性是 transform: translateY(-100px);

          而且 transform 不仅可以设置水平和垂直方向的位移, 还有角度的旋转也是用 transform 来设置

          
width:      控件宽度的变化
height:     控件高度的变化
opacity:    控件透明度的变化
background: 背景颜色的变化

位置的变化:
transform:  对控件的位置,水平垂直位移还有角度变化, 都使用transform来设置

          transition 动画的触发 ,transition 动画的定义本质就是: 指定了什么属性发生变化, 则怎么进行过渡, 于是触发实际上就是在定义好了 transition 动画后, 只需要改变相应的 css 样式即可触发动画, 那么对控件的属性进行改变的方法, 一般都需要使用 js ,
          比如, 用户点击按钮后, 我们就对某个控件的透明度进行改变; 比如, 当用户滚动到某个区域, 我们就对某个 css 属性进行改变; 于是属性相应的动画得到运行

          这里我们举例子有个简单的方式, 就是使用 hover 这个伪类, 当控件被 hover 的时候, 我们就给它设置不同的 css 属性值, 从而来触发 transition 动画

          # demo1 : 针对一个 css 属性的 transition 动画

          注意:

          在下面例子中, 如果 transition 动画定义在.box 中, 那么从普通状态到 hover 状态的 background 变化都有过渡的动画效果

          但是如果, 把 transition 动画定义在.box:hover 中, 那么从普通状态到 hover 状态 background 变化是有过渡动画效果的, 但是从 hover 到普通状态是没有动画的

          所以, 如果从属性值 A 到 B 的动画效果和 B 到 A 的动画效果不同的话, 那么就需要写不同的动画

          属性值 A 到属性值 B 的动画 , 应该写在 B 的样式里

          属性值 B 变化到 A 动画应该写在 A 样式里

          // 下面这个例子, 从正常状态转变到 hover 状态 过渡时间是 500ms // 从 hover 状态过渡到正常状态是 2 秒 // 例 1

          <!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <meta http-equiv="X-UA-Compatible" content="ie=edge" />
    <title>Document</title>
  </head>
  <body>
    <div class="box"></div>

    <style>
      .box {
        width: 300px;
        height: 300px;
        background: red;
        transition-property: background;
        transition-duration: 2000ms;
        transition-timing-function: ease-in-out;
      }

      .box:hover {
        background: blue;
        transition-property: background;
        transition-duration: 500ms;
        transition-timing-function: ease-in-out;
      }
    </style>
  </body>
</html>

          # demo2 两个属性值的变化动画

          // 例 2 两个属性值的变化动画 // 只需要设置 transition-property: 为多个 css 样式即可 // 但是这样设置的时候, 多个 css 属性使用的是相同的动画效果

          <style>
  .box {
    width: 300px;
    height: 300px;
    background: red;
    transition-property: background, transform;
    transition-duration: 500ms;
    transition-timing-function: ease-in-out;
  }

  .box:hover {
    background: blue;
    transform: rotate(45deg);
  }
</style>

          # demo3 : 针对多个 css 样式设置不同的过渡动画效果

          // 例 3: 针对多个 css 样式设置不同的过渡动画效果 // 必须使用简写, 不使用简写的话, // transition 动画属性无法针对不同属性设置不同值, 因为 css 属性值没有次序之分, // 后面的属性值会覆盖前面的值, 所以针对不同属性设置不同的动画, 只能使用简写方式

          // background 变化立即执行,过渡时间 1 秒 // transform 转动延时 1 秒后执行, 过渡时间 500ms

          <style>
  .box {
    width: 300px;
    height: 300px;
    display: flex;
    justify-content: center;
    align-items: center;
    background: red;
    transition: 1s background ease-in, 500ms transform ease-out 1s;
  }

  .box:hover {
    background: blue;
    transform: rotate(45deg);
  }
</style>

          # transition 简写

          transition 过渡时间 css 属性 变化率名称 执行延迟时间

          transition: 1s background ease-in, 500ms transform ease-out 1s;

          比如 // background 变化立即执行,过渡时间 1 秒 // transform 转动延时 1 秒后执行, 过渡时间 500ms
